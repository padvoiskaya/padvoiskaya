## 🌐 Тестирование веб-приложений 

Изучила основы работы веб-приложений и принципы их тестирования:

- Принципы работы Internet: IP, MAC, URL, DNS.
- Клиент-серверная архитектура: Монолиты, микросервисы, балансировщики.
- HTTP-протокол: Методы, коды ответов, запросы и ответы.
- Кэш и куки: Влияние на поведение приложений.
- Различия между веб-сайтом, веб-приложением, веб-сервисом и сервером.
- Инструменты Chrome DevTools для отладки и тестирования.
- Основы HTML и CSS для тестирования UI. Виды UI и верстки.

🔧 Результаты в репозитории [Тестирование веб-приложений](https://github.com/padvoiskaya/web)
